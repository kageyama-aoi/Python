# ファイル整理ツール

これは、指定されたディレクトリ内のファイルを拡張子ごとにサブディレクトリを作成して整理するPythonスクリプトです。

## 機能

- 指定したディレクトリ内のファイルを、そのファイルの拡張子と同じ名前のフォルダに移動します。
- 拡張子がないファイルは `no_extension` という名前のフォルダに移動します。
- 整理の実行ログを、対象ディレクトリ内の `logs` フォルダに保存します。
- 移動先に同名ファイルが存在する場合は、ファイル名の末尾に連番を付与してリネームします。

## 使い方

1. **設定ファイルの編集**
   - `config.ini` を開き、`TargetDirectory` に整理したいフォルダのパスを指定します。
   - デフォルトでは `C:\Users\kageyama\OneDrive\Desktop` に設定されています。

2. **スクリプトの実行**
   - `run_organizer.bat` をダブルクリックして実行します。

## ファイル構成

- `organize_files.py`: ファイル整理処理のメインスクリプトです。
- `config.ini`: 整理対象のディレクトリなどを設定するファイルです。
- `run_organizer.bat`: `organize_files.py` を実行するためのバッチファイルです。
- `README.md`: このファイルです。
