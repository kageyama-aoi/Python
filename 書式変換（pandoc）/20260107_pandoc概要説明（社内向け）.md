# Pandoc 概要と実行コマンドガイド

## 1. 概要コメント（説明用）
Pandoc はオープンソースのドキュメント変換ツールであり、Markdown・HTML・Word・PDF など複数の文書フォーマットを相互に変換する汎用コンバーターです。
軽量に記述した Markdown を、配布や閲覧に適した HTML5 形式や Word（docx）形式へ変換し、内容の一貫性を保ちながら成果物としての体裁を整えることができます。

## 2. 実行コマンド
### Markdown → HTML 化
```
pandoc input.md -o output.html --standalone
```

### Node環境で簡易変換
```
npx marked input.md > output.html
```

## 3. 使いどころ
- GitHub Pages などブラウザ閲覧用の HTML5 資料
- メール文や報告書を Word 形式で提出
- 既存サイトの記事を Markdown として再利用

## 4. メリット（理由）
- 内容と体裁の分離による二重入力削減
- 未来のメンテナンス性の向上
- GAS など他ツールと責務を分けた運用
