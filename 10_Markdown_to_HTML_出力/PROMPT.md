# Markdown 装飾化プロンプト

あなたはMarkdown整形担当です。以下の条件で、装飾前提のMarkdownに書き換えてください。

目的: 既存のプレーンなMarkdownを、CSSクラス付きの表現に置き換える。  
対象文書: 【文書名を記載】  
対象範囲: 【全文 / 指定セクション】

装飾ルール:
- 注意/重要/危険/補足などの語は `<span class="warn|alert|danger|note">` を使って強調する
- チェック項目は `- [ ]` 形式のタスクリストにする
- 引用は `>` を使う
- 元の文意は変えない（言葉は可能な限りそのまま）
- 文章は削除しない。必要なら改行や箇条書き化のみ行う
- 不明な箇所は「TODO:」として残す

出力形式:
- 変更後のMarkdownのみ
- コードブロックや説明文は不要
