# ツール名称とMarkdown効率化案

## 呼び方の候補

- ナレッジハンドブック
- 運用ナレッジベース
- Markdownハンドブック
- 社内Wiki（Markdown版）

## Markdown記法の手間を減らす方法

### 1. テンプレート化

- `docs/template.md` を作る
- よく使う埋め込み・リンクの雛形を用意する
- 新規作成時はテンプレートをコピーして使う

### 2. スニペット化（VS Code）

- ユーザースニペットに登録して `Tab` 展開する
- リンク用:
  - `[${1:text}](${2:path})`
- 画像用:
  - `![${1:alt}](${2:path})`

### 3. 補助ツール化

- このリポジトリに `md_helper.ps1` を作る
- ファイルパス指定でリンク文字列を自動生成する
- 画像リンクや相対パスも自動化できる
