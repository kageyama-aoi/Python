# コミットメッセージ提案用 指示文

このファイルを読み込んだうえで、以下の指示文を実行してください。

```text
このリポジトリのREADMEおよび関連ドキュメントの方針に従って、`git diff --staged` の内容からコミットメッセージ案を作成してください。

要件:
- Conventional Commits形式
- 日本語
- 3案
- 各案は1行のみ
- 形式は `type(scope): summary`
- ニュアンスを少しずつ変える
- 解説は書かない

出力形式:
1. ...
2. ...
3. ...
```

## 実行前提

- 先に `git add` を済ませて、`git diff --staged` に差分がある状態にすること。

## READMEに書くときの短い文例

```text
コミットメッセージを提案させるときは、`COMMIT_MESSAGE_INSTRUCTIONS.md` を読み込んでから実行してください。
```
